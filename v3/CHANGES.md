# 변경사항 정리 (v2 → v3)

## 전반적인 변경사항

1. 구조적 변경
   - 스크립트를 목적별로 3개의 디렉토리로 분리
     * 24x7/: 1차 기술지원용 기본 점검
     * platform/: 2차 기술지원용 플랫폼 점검
     * pod/: 2차 기술지원용 파드 점검

2. 보안 강화
   - 민감 정보 마스킹 기능 추가
     * UUID, Base64 인코딩 문자열
     * Client ID/Secret
     * Subscription/Tenant ID
     * SSH 키
     * 패스워드, 시크릿, 토큰, API 키
     * Managed Identity, Principal ID
     * Connection 문자열
     * Bearer 토큰

3. 로깅 개선
   - 구조화된 로그 출력 형식 도입
   - 타임스탬프 기반 로그 파일명
   - 실시간 화면 출력과 파일 로깅 동시 지원

## 스크립트별 주요 변경사항

### 1. 24x7 점검 스크립트 (24x7_check_aks.sh → 24x7_aks_check.sh)

#### 기능 추가
- 사전 요구사항 검사 (az, kubectl, jq)
- Azure 로그인 상태 확인
- 클러스터 전원 상태 확인
- 비정상 파드 자동 감지 및 경고

#### 기능 단순화
- 기본적인 상태 점검에 집중
- 복잡한 쿼리 제거
- 직관적인 결과 표시

### 2. 플랫폼 점검 스크립트 (check_aks.sh → aks_check.sh)

#### 기능 추가
- 클러스터 구성 상세 점검
- 노드풀 자동 스케일링 설정 확인
- 네트워크 연결성 테스트
- 스토리지 구성 상세 점검
- 보안 설정 검사 (RBAC, PSP)
- 모니터링 설정 검사

#### 기능 개선
- 체계적인 메트릭 수집
- 상세한 보안 점검
- 진단 설정 확인

### 3. 파드 점검 스크립트 (신규: aks_pod.sh)

#### 주요 기능
- 네임스페이스별 파드 상태 점검
- 파드 리소스 사용량 모니터링
- 파드 네트워킹 상태 확인
- 볼륨 마운트 상태 확인
- 파드 로그 분석
- 이벤트 모니터링
- 파드 구성 검증

#### 특징
- 단일/전체 네임스페이스 점검 지원
- 상세한 문제 진단 정보 제공
- 자동화된 로그 분석

## 환경 설정 개선

1. 환경 변수 파일 분리
   - 24x7_aks_check.env

   - aks_pod.env

2. 설정 항목 확장
   - 리소스 그룹

   - 구독 ID
   - 위치 정보
   - 모니터링 설정
   - 네임스페이스 설정 (선택적)

## 사용성 개선

1. 체계적인 에러 처리
   - 명확한 에러 메시지
   - 실패 원인 식별 용이
   - 문제 해결 가이드 제공

2. 모듈화된 구조
   - 기능별 함수 분리
   - 재사용 가능한 유틸리티 함수
   - 유지보수 용이성 향상

3. 확장성
   - 새로운 점검 항목 추가 용이
   - 커스터마이징 가능한 구조
   - 모듈식 기능 확장

## 점검항목 변경사항

### 1. 24x7 점검 스크립트 점검항목

#### 기존 항목 유지
- Azure Kubernetes 버전 확인
- 네트워크 정책(NetworkPolicy) 확인
- Pod 상태 확인
- 시스템 서비스 상태 확인

#### 새로 추가된 항목
- 사전 요구사항 점검 (필수 도구 설치 여부)
- Azure 로그인 상태 점검
- 클러스터 프로비저닝 상태 점검
- 클러스터 전원 상태 점검
- 노드 상태 상세 점검
- 비정상 파드 자동 감지

### 2. 플랫폼 점검 스크립트 점검항목

#### 기존 항목 유지
- 네트워크 정책 확인
- Keyvault 통합 확인
- Namespace 상태 확인
- Pod 상태 및 사용량 확인
- 서비스 및 엔드포인트 확인
- PVC 상태 확인
- Storage Account 연결 상태
- Azure Monitor 활성화 확인

#### 새로 추가된 항목
- 클러스터 구성 상세 점검
  * 네트워크 플러그인 설정
  * 네트워크 정책 설정
  * 애드온 프로필 설정
  * API 서버 접근성

- 노드풀 상세 점검
  * 자동 스케일링 설정
  * 노드 리소스 상태
  * 노드 버전 관리
  * 노드 풀 구성

- 네트워크 연결성 점검
  * DNS 해석 테스트
  * 서비스 연결성
  * 네트워크 정책 효과

- 스토리지 상세 점검
  * 스토리지 클래스 설정
  * PV/PVC 상태
  * 스토리지 프로비저너

- 보안 설정 점검
  * RBAC 구성
  * Pod Security Policies
  * 서비스 계정 설정

- 모니터링 상세 점검
  * 진단 설정
  * 메트릭 수집
  * 로그 수집
  * Prometheus 통합

### 3. 파드 점검 스크립트 점검항목 (신규)

#### 파드 상태 점검
- 파드 실행 상태
- 재시작 횟수
- 파드 스케줄링 상태
- 파드 이벤트 로그

#### 리소스 사용량 점검
- CPU 사용량
- 메모리 사용량
- 스토리지 사용량
- 리소스 제한 설정

#### 네트워킹 점검
- 서비스 연결
- 엔드포인트 상태
- 네트워크 정책 적용
- DNS 해석

#### 스토리지 점검
- 볼륨 마운트 상태
- PVC 바인딩 상태
- 스토리지 용량
- 스토리지 접근성

#### 로그 및 이벤트 점검
- 애플리케이션 로그
- 시스템 로그
- 이벤트 기록
- 에러 패턴 분석

#### 구성 점검
- 디플로이먼트 설정
- 레플리카셋 상태
- 컨피그맵 설정
- 시크릿 설정
