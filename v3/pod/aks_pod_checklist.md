## 스크립트 설명

`aks_pod.sh` 스크립트는 AKS 클러스터의 파드 수준 상세 점검을 수행하는 스크립트입니다. 이 스크립트는 다음을 수행합니다.

*   특정 네임스페이스 또는 모든 네임스페이스의 파드 상태 확인
*   파드 리소스 사용량 확인
*   파드 네트워킹 확인 (서비스, 엔드포인트, 네트워크 정책)
*   파드 볼륨 확인 (PVC, 볼륨 마운트)
*   파드 로그 확인 (최근 50줄, 에러 패턴)
*   파드 이벤트 확인
*   파드 구성 확인 (디플로이먼트, 레플리카셋, 컨피그맵, 시크릿)

이 스크립트는 로그 파일에 결과를 저장합니다.

## 확인 방법

1.  `aks_pod.env` 파일에 점검 대상 AKS 클러스터의 `RESOURCE_GROUP`과 `CLUSTER_NAME`을 설정합니다. 특정 네임스페이스만 점검하려면 `NAMESPACE` 변수를 설정합니다.
2.  스크립트를 실행합니다: `bash ./aks_pod.sh`
3.  스크립트 실행 결과는 터미널과 `./logs/aks_pod_*.log` 파일에 출력됩니다.
4.  로그 파일에서 각 점검 항목의 결과를 확인합니다.
    *   **파드 상태 확인**: 각 네임스페이스의 파드 목록과 상태를 확인합니다. `Running` 또는 `Completed`가 아닌 파드가 있는지, 있다면 해당 파드의 `describe` 정보와 로그를 확인합니다.
    *   **파드 리소스 사용량 확인**: `kubectl top pods` 결과로 각 파드의 리소스 사용량을 확인하고, `describe` 결과로 리소스 제한 설정을 확인합니다.
    *   **파드 네트워킹 확인**: 서비스, 엔드포인트, 네트워크 정책 목록을 확인합니다.
    *   **파드 볼륨 확인**: PVC 목록과 각 파드의 볼륨 마운트 정보를 확인합니다.
    *   **파드 로그 확인**: 각 파드의 최근 로그 50줄과 에러 관련 로그를 확인합니다.
    *   **파드 이벤트 확인**: 시간순으로 정렬된 파드 이벤트를 확인하여 문제 발생 시점을 추적합니다.
    *   **파드 구성 확인**: 디플로이먼트, 레플리카셋, 컨피그맵, 시크릿 목록을 확인합니다.
